# Life Choices - Backend API

Ова е бекенд серверот за проектот **Life Choices**, изграден со Node.js и Express. Серверот овозможува интеграција со Groq API за процесирање и генерирање податоци преку вештачка интелигенција.

## Технологии
* **Node.js** (извршна средина)
* **Express** (веб рамка за API рути)
* **Groq SDK** (интеграција со AI модел 'llama-3.3-70b-versatile')
* **Cors** (овозможува безбедни барања на фронтендот)
* **Dotenv** (менаџирање на конфигурациски и околински променливи)

## Локално стартување

Со цел успешно да ја стартувате апликацијата, следете ги следниве чекори на вашиот локален компјутер.

### 1. Инсталација на зависности (Dependencies)
Инсталирајте ги потребните пакети преку npm 

```
npm install
```

### 2. Конфигурација на околина (.env)
Во коренот на бекенд директориумот, креирана е датотека со име .env каде е потребно да го ставите вашиот API клуч. Исто така во оваа датотека е дефинирана и портата на која ќе слуша серверот.

### 3. Покренување на серверот

За да го пуштите серверот во работа извршете ја командата:

```
npm start
```

Апликацијата ќе започне да работи на **http://localhost:3001**.

## API Ендпоинти (Endpoints)

### 1. Проверка на работата на серверот (Health Check)
* URL: ```/```
* Метод: ```GET```
* Опис: Враќа статус со кој се потврдува дека API-то работи без проблем.
* Одговор (JSON):
```
{
  "status": "Life Choices API is running ✅"
}
```

### 2. Генерирање на AI резиме за благосостојба
* URL: ```/api/summary```
* Метод: ```POST```
* Опис: Прифаќа податоци за финалните статистики на корисникот и историјата на неговите избори, па генерира топол, мотивациски и контекстуален текст со помош на AI.
* Тело на барањето (Request Body - JSON):
```
{
  "stats": {
    "sleep": 80,
    "mood": 75,
    "social": 60,
    "focus": 90
  },
  "choiceHistory": [
    { "scenario": "Спиење пред тест", "good": true },
    { "scenario": "Исхрана и благо", "good": false }
  ]
}
```
* Одговор (JSON):
```
{
  "summary": "Браво! Твојот фокус и квалитет на сон се на навистина високо ниво..."
}
```
