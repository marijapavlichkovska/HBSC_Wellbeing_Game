# Life Choices — HBSC Wellbeing Game

Интерактивна едукативна игра базирана на реални податоци од **HBSC студијата (Health Behaviour in School-aged Children)**, наменета за адолесценти на возраст 11–15 години од Македонија.

---

## 1. Истражување и идентификуван проблем

HBSC студијата (2026) открива повеќе загрижувачки трендови кај македонските тинејџери:

* **42%** од 15-годишниците се чувствуваат уморни во училиште секој ден поради доцно користење на екрани
* **Само 30%** спијат препорачаните 8–10 часа навечер
* **Само 18%** ги исполнуваат СЗО препораките за 60 минути дневна физичка активност
* **1 од 4** тинејџери се чувствува осамено во поголем дел од времето
* **Само 1 од 3** адолесценти кои имаат потешкотии со менталното здравје бара помош

Овие податоци укажуваат на системски недостаток на свесност кај младите за тоа **како секојдневните одлуки директно влијаат врз нивната благосостојба** — спиење, расположение, социјалност и концентрација.

---

## 2. Дефинирање на идеја

### Проблем
Македонските адолесценти не се доволно свесни за врската помеѓу нивните секојдневни избори и менталното/физичкото здравје. Традиционалните едукативни пристапи (предавања, брошури) не го привлекуваат вниманието на оваа возрасна група.

### Целна група
Адолесценти на возраст **11–15 години** (соодветно на HBSC таргет популацијата), особено ученици во средно образование во Македонија.

### Решение
**Life Choices** е сценарио-базирана веб-игра во која корисникот ја управува постапката на **Alex**, 15-годишен тинејџер од Скопје. Низ 12 реалистични животни ситуации: од доцно скролање пред спиење до притисок од врсници за пушење, играчот носи одлуки кои ги менуваат четири статистики на благосостојба: **Сон, Расположение, Социјалност и Фокус**.

По секој избор, играчот добива:
* **Моментален feedback** со емоционална порака
* **HBSC факт** поврзан со Македонија и реални статистики
* **Видливи промени** во статистиките (визуелни ленти)

На крајот, **AI wellbeing coach** (Groq / LLaMA 3.3) генерира персонализирано резиме базирано на изборите на играчот.

### Очекуван impact
* Зголемена свесност кај младите за последиците од секојдневните одлуки
* Смалување на стигмата околу менталното здравје преку нарацијата на Alex
* Поттикнување на разговор помеѓу ученици, наставници и родители за темите опфатени во играта

## 3. Техничка реализација

### Архитектура

Проектот е поделен на два дела:

```
HBSC_Wellbeing_Game/
├── life-choices/           # Frontend — React апликација
└── life-choices-backend/   # Backend — Node.js + Express API
```

### Технологии

| Слој | Технологија | Намена |
|------|-------------|--------|
| Frontend | React 19 | UI, логика на играта, state management |
| Frontend | CSS-in-JS (styles.js) | Целосен визуелен дизајн |
| Backend | Node.js + Express | REST API сервер |
| Backend | Groq SDK (LLaMA 3.3-70b) | Генерирање на AI wellbeing резиме |
| Backend | dotenv, cors | Конфигурација и безбедност |

### UI/UX Дизајн

Апликацијата е организирана во четири јасно дефинирани екрани:

1. **Intro Screen** — Воведна страница со опис и HBSC контекст
2. **Game Screen** — Сценарио со три избори + живи статистички ленти
3. **Feedback Screen** — Последици од изборот + HBSC факт + delta индикатори
4. **End Screen** — Финални статистики + персонализирано AI резиме

### Логика на функционирање

* Играчот почнува со **70 поени** во секоја од четирите статистики
* Секој избор носи `deltas` (позитивни или негативни промени) на статистиките
* Вредностите се заштитени со `clamp(5, 100)` — не можат да излезат надвор од опсегот
* По сите 12 сценарија, финалните вредности и историјата на избори се испраќаат до бекендот
* Бекендот го конструира prompt за AI моделот и враќа персонализиран текст

### Сценарија

Играта опфаќа 12 тематски области директно поврзани со HBSC податоците:

🌙 Sleep & Screens · 📚 School Stress · 💬 Cyberbullying · 👥 Peer Pressure · 🏃 Physical Activity · 💭 Loneliness · 🍔 Eating Habits · 🚬 Substance Pressure · 😤 Family Conflict · 🎮 Screen Time · 🧠 Mental Health · 🌍 Social Causes

## 4. Локално стартување

### Предуслови

* Node.js (v18 или понова верзија)
* npm
* Groq API клуч (бесплатен на [console.groq.com](https://console.groq.com))

### Чекори

#### 1. Клонирање на репозиториумот

```
git clone https://github.com/your-username/HBSC_Wellbeing_Game.git
cd HBSC_Wellbeing_Game
```

#### 2. Стартување на бекендот

```
cd life-choices-backend
npm install
```

Во датотеката `.env` додајте го вашиот Groq API клуч:

```
GROQ_API_KEY=enter_your_groq_api_key_here
PORT=3001
```

```
npm start
```

Бекендот ќе работи на **http://localhost:3001**.

#### 3. Стартување на фронтендот

Во нов терминал:

```
cd life-choices
npm install
npm start
```

Апликацијата ќе работи на **http://localhost:3000**.

> ⚠️ И фронтендот и бекендот мора да работат истовремено за целосна функционалност на AI резимето.
